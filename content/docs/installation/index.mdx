## Быстрая установка (Рекомендуется)

### Linux / macOS

```bash
curl -fsSL https://get.razd-cli.com/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm https://get.razd-cli.com/install.ps1 | iex
```

> **Альтернатива:** Если короткий URL недоступен, используйте прямые ссылки GitHub:
> - Linux/macOS: `curl -fsSL https://raw.githubusercontent.com/razd-cli/installer/main/install.sh | bash`
> - Windows: `irm https://raw.githubusercontent.com/razd-cli/installer/main/install.ps1 | iex`

### Установка конкретной версии

**Linux / macOS:**

```bash
# Установить конкретную версию через переменную окружения
RAZD_VERSION=1.0.0 curl -fsSL https://get.razd-cli.com/install.sh | bash

# Или при запуске напрямую:
./install.sh --version 1.0.0
./install.sh --pre-release        # последняя pre-release версия
```

**Windows (PowerShell):**

```powershell
# Установить конкретную версию через переменную окружения
$env:RAZD_VERSION = "1.0.0"; irm https://get.razd-cli.com/install.ps1 | iex

# Или при запуске напрямую:
.\install.ps1 -Version 1.0.0
.\install.ps1 -PreRelease
```

### Пользовательский каталог установки

**Linux / macOS:**

```bash
RAZD_INSTALL_DIR=/usr/local/bin curl -fsSL https://get.razd-cli.com/install.sh | bash
# или
./install.sh --dir /usr/local/bin
```

**Windows (PowerShell):**

```powershell
$env:RAZD_INSTALL_DIR = "C:\Tools\razd"; irm https://get.razd-cli.com/install.ps1 | iex
# или
.\install.ps1 -InstallDir "C:\Tools\razd"
```

## Установка через mise

Также можно установить Razd через mise — менеджер инструментов разработки:

### Установка mise

**Unix/Linux/macOS:**

```bash
curl https://mise.run | sh
```

**macOS (через Homebrew):**

```bash
brew install mise
```

**Windows:**

```powershell
winget install jdx.mise
```

> После установки mise перезапустите терминал или выполните `source ~/.bashrc` (Linux) / `source ~/.zshrc` (macOS), чтобы обновить PATH.

### Установка Razd через mise

```bash
# Установите плагин Razd
mise plugin install razd https://github.com/razd-cli/vfox-plugin-razd

# Установите и используйте последнюю версию глобально
mise use -g razd@latest

# Проверьте установку
razd --version
```

### Установка конкретной версии через mise

```bash
# Установите конкретную версию
mise use -g razd@0.4.4

# Или установите локально для проекта
cd my-project
mise use razd@0.4.4
```

## Проверка установки

После установки любым методом проверьте корректность работы:

```bash
# Проверьте версию
razd --version

# Проверьте справку
razd --help
```

## Обновление Razd

### Обновление через инсталлер

**Linux / macOS:**

```bash
curl -fsSL https://get.razd-cli.com/install.sh | bash
```

**Windows (PowerShell):**

```powershell
irm https://get.razd-cli.com/install.ps1 | iex
```

### Обновление через mise

```bash
# Обновите до последней версии
mise use -g razd@latest

# Или до конкретной версии
mise use -g razd@0.4.4
```

## Удаление

### Удаление при установке через инсталлер

Удалите бинарный файл:

**Linux / macOS:**

```bash
rm ~/.local/bin/razd
```

**Windows:**

```powershell
Remove-Item "$env:LOCALAPPDATA\razd\razd.exe"
```

### Удаление через mise

```bash
# Удалите Razd
mise uninstall razd

# Удалите плагин (опционально)
mise plugin uninstall razd
```

### Ошибка: "mise not found"

Установите mise, следуя инструкциям в разделе [Предварительные требования](#предварительные-требования).

## Дополнительные инструменты

### VS Code расширение (Рекомендуется)

Для удобной работы с Razd в Visual Studio Code установите официальное расширение:

**Установка через VS Code:**

1. Откройте VS Code
2. Перейдите в раздел расширений (Ctrl+Shift+X)
3. Найдите "Razd" или используйте ID: `razd-cli.vscode-razd`
4. Нажмите "Install"

**Установка через командную строку:**

```bash
code --install-extension razd-cli.vscode-razd
```

**Ссылка на Marketplace:** [https://marketplace.visualstudio.com/items?itemName=razd-cli.vscode-razd](https://marketplace.visualstudio.com/items?itemName=razd-cli.vscode-razd)

Расширение предоставляет удобства при работе с Razdfile.

### task (Опционально, устанавливается автоматически)

Task используется для запуска задач проекта. Если он отсутствует, Razd установит его через mise:

```bash
# Ручная установка (опционально)
mise use -g task@latest
```
