## Скоро появятся

- **Python** - настройка проектов на Python с pip, poetry, и виртуальными окружениями
- **Rust** - настройка Rust проектов с cargo и rustup
- **Go** - настройка Go проектов с модулями и инструментами
- **Ruby** - настройка Ruby проектов с bundler и rbenv
